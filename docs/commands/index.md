## Utility Commands

### `^help`

See commands that you can issue.

**Usage:** `^help [command/category]`

**Aliases:** `^?`

**Parameters:**

* command/category - Command or category to get help with

## Moderation Commands

### `^warn`

Warn a user and mark an infraction on their account.

**Usage**: `^warn (user) (reason)`
